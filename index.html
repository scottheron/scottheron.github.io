Cards Against the World

HTML | CSS | Bootstrap | JavaScript | jQuery | jQuery UI | jQuery Flip | Font Awesome

Cards Against The World is a clone of the popular adult card game Cards Against Humanity. It has a reduced set of cards and is limited to two players and a PC player who contributes cards to simulate a four player game.

The Whisky Stone

HTML | CSS | Materialize | JavaScript | jQuery | Node | Express | JDOM | Flickr

The Whisky Stone is an application that allows a user to search for single malt Scottish Whisky, browse available bottlings and read the tasting notes to allow them to make the perfect selection before they head to the store. The Whisky data is scrapped from a Whisky database.

About
