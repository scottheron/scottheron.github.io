Cards Against The World

HTML | CSS | Bootstrap | JavaScript | jQuery | jQuery UI | jQuery Flip | Font Awesome

Cards against The World is a clone of the popular adult card game of the same name. It has a reduced set of cards and is limited to two players and a PC player who contributes cards to simulate a four player game. It was written as the first project on General Assembly's WDI course in Seattle.

The Whisky Stone

HTML | CSS | Materialize | JavaScript | jQuery | Node | Express | Postgresql | JDOM | Flickr

The Whisky Stone was the second project from the General Assembly WDI course. It's an application that allows a user to create a profile, search for single malt Scottish whisky and create a list of their favorites. They can tag the whisky with key words which are visable to other users. The Whisky data was scrapped from a Whisky database and presented for the user to explore. Information on the Whisky included tasting notes and bottling years.

Rainy Day

HTML | CSS | Bootstrap | Ruby | Rails | Postgresql | Sequelize | Google Places

Rainy Day was the third project from the General Assembly WDI course. It's an application to help users find places they havent been to and save them to a profile, creating a list of places to visit when they have spare time. In addition they can add goals to acomplish at each location, like climb to the top of snake pass. Once they accomplish their goals they can unlock achievements which are listed on their profile.

The Wizards Unicorn

HTML | CSS | Bootstrap | JavaScript | Angular | Node | Express | MongoDB | Microsoft Voice to Text

The Wizards Unicorn was the fourth project from the General Assembly WDI course. The app is a text adventure game controlled by the users voice. The user can play by speaking in to a mic or by typing commands if the API can't recognize their speech. The text adventure was written using JavaScript and Angular. Microsoft's Voice to Text api was used to convert the users voice in to text.
